Нам осталось разобраться, как сделать игру шахматы в веб-браузере. Мы сделаем это с помощью модуля PySimpleGUIWeb. Установить модуль можно командой: 

terminal
    pip install PySimpleGUIWeb

Возьмем код, который мы написали при создании приложения с помощью модуля PySimpleGUI (тема "ChessGUI") и внесем в него несколько изменений. Поменяем импорт. 

python
    # import PySimpleGUI as sg
    import PySimpleGUIWeb as sg

Далее один из цветов я поменяю на коричневый, т.к. не все цвета поддерживаются браузерами. 

python
    # COLORS = ('gold', 'coral3', 'gray85')
    COLORS = ('gold', 'brown', 'gray85')

Так же внесем изменения в аргумент text_color в def play() и списке layout. 

python
    # text_color='gray' if not value or value.isalnum()

    text_color='white' if value and 9811 < ord(value) < 9818 else 'black')

Удалим из кода команды, которые пока не поддерживаются модулем. 

python
